BulletSurvivor

  쏟아지는 총알과 몰려오는 몬스터들로부터 살아남는 탑다운 슈팅 서바이벌 게임입니다.

  🎮 게임 소개

  BulletSurvivor는 플레이어가 끊임없이 몰려오는 몬스터들을 상대로 생존해야 하는 액션 로그라이크 게임입니다. 다양한
  무기를 획득하고 캐릭터를 강화하여 최후의 생존자가 되세요.

  ✨ 주요 기능

   - 끊임없이 몰려오는 몬스터 웨이브
   - 다양한 무기 획득 및 강화 시스템
   - 플레이어 체력(HP) 및 상태 UI
   - 몬스터 처치 수(킬 카운트) 기록
   - 게임 오버 및 재시작/타이틀 이동 기능

  🕹️ 조작법

   - 이동: W, A, S, D
   - 조준 및 발사: 마우스 커서 및 클릭

  🛠️ 설치 및 실행 방법

  빌드된 게임 실행

   1. 이 저장소를 다운로드하거나 복제합니다.
   2. Build/ 폴더로 이동합니다.
   3. BulletSurvivor.exe 파일을 실행합니다.

  Unity에서 프로젝트 열기

  이 프로젝트는 Unity로 개발되었습니다. 프로젝트를 수정하거나 빌드하려면 Unity Hub에서 '기존 프로젝트 추가'를 선택하고
  이 프로젝트의 루트 폴더를 지정하세요.

   - 권장 Unity 버전: ProjectSettings/ProjectVersion.txt 파일에서 확인된 버전

  💻 사용된 기술

   - 게임 엔진: Unity
   - 프로그래밍 언어: C#
